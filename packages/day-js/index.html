dayjs

Day.js module for Nuxt

Installation 

npm i -D dayjs-nuxt

Setup

Add the following lines to the modules section of nuxt.config.{ts,js} 

modules: ['dayjs-nuxt']

Demo

DayJs.vue
<script setup>
const dayjs = useDayjs()
const dateTime = computed(() => dayjs().format('DD MMM, YYYY') )
</script>

<template>
  <span>
    {{ dateTime }}
  </span>
</template>
14 Jul, 2024
